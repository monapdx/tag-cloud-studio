# Tag Cloud Studio

Interactive tag/word cloud studio for building, styling, and exporting beautiful word clouds.

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg) ![PRs welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)

## Features
- Import word lists (CSV/JSON/plain text)
- Style fonts, colors, shapes, and layouts
- Live preview and export (PNG/SVG)
- Adjustable stop-words and weights
- Fast dev experience with modern tooling

## Quick Start

### 1) Clone
```bash
git clone https://github.com/your-username/tag-cloud-studio.git
cd tag-cloud-studio
```

### 2) Install
```bash
npm install
```

### 3) Run
```bash
npm run dev  # or: npm start
Open the URL printed by Vite (e.g., http://localhost:5173).
Tailwind CSS is preconfigured; adjust `tailwind.config.*`.
```

### 4) Build
```bash
npm run build
```

## Scripts
- **dev** — `vite`
- **build** — `vite build`
- **preview** — `vite preview`
